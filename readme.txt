== Description ==

Slide View plugin for [Lazyest Gallery](http://wordpress.org/extend/plugins/lazyest-gallery/)

This plugin adds new slide views to your Lazyest Gallery
Requires at least Lazyest Galery 1.1.1

= In version 0.5 =
* Slide view with thumbs slider
* Thickbox popup with caption and description

== Installation ==

1. Install using the WordPress plugin installer
2. Activate 
3. Click on the Setup Slides button in Settings -> Lazyest Gallery -> Slide View Options -> Alternative Display
